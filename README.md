API
---

always 303 redirect:

* `http://npm-tarball.antle.red/:package`

  e.g.: [Latest Jquery](http://npm-tarball.antle.red/jquery)

* `http://npm-tarball.antle.red/:package/:semver`

  e.g.: [v1 Jquery](http://npm-tarball.antle.red/jquery/1)
